## lab7-vectormem
## Vector Calculator V2
# This program allows the user to perform math operations on vectors, using dynamci memory to store them.

# To build:
make clean
make

# To run:
./minimat

# Supported commands:
add
subtract
multiply a vector by a number
display answer or save answer into a new vector
list
help
display a vector
clear saved vectors
quit
